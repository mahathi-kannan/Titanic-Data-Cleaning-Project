## Project Overview
This project involves cleaning and analyzing the Titanic dataset to discover patterns in passenger survival. It was completed as Task 1 of my Data Science Internship.

## Steps Taken:
- **Data Cleaning:** Handled 86 missing values in 'Age' and 1 in 'Fare' using median imputation.
- **Data Preprocessing:** Dropped the 'Cabin' column due to excessive missing data (over 70%).
- **Visualization:** Created charts using Seaborn to analyze survival rates based on Gender, Passenger Class, and Age.

## Task 3: Exploratory Data Analysis (EDA)
In this phase, I acted as a data detective to uncover the "why" behind the survival rates.

### Key Insights:
* **Correlation:** Higher fares and lower class numbers (1st Class) significantly increased survival chances.
* **Demographics:** Age and Gender remained the most dominant factors in the decision-making process during the disaster.
* **Family Size:** Traveling with a small family (1-3 people) showed a better survival trend than traveling alone.
